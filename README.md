# Installation Steps

## Using npm

Run commands

1) ```npm install```


2) ```npm start```

3) ```press a on keyboard``` 


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn  start```

4) ```press a on keyboard```
